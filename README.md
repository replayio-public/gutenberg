## Instructions

Getting started

1. `npm install`
2. `npm run build`
3. `npm run wp-env start`

Running the tests locally without Replay

1. `npm run test:e2e:playwright`

Running the tests locally with Replay Chromium

1. `npm run test:e2e:playwright:replay`
